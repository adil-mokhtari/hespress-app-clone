# 📰 Hespress App Clone

This is a Flutter-based news app that connects to a custom WordPress plugin. It supports Arabic (RTL), categories, search, and navigation drawer UI.

## 🔧 Features

- 🗞️ Latest news from WordPress
- 🧭 Navigation drawer
- 🔍 Search page
- 📚 Categories filter
- 📄 Post detail page
- 🌐 RTL Arabic support
